# Set & Rep Configuration Schemes — Research Findings (2026-06-26)

Deep-research run `wf_638a542c-05d` (107 agents, 25 sources, 115 claims → 25 adversarially verified, 20 confirmed / 5 refuted). Use this as the citable basis for the set-scheme engine. Confidence tiers below distinguish RCT-verified mappings from coaching consensus.

## Verified (RCT / meta-analysis)

- **Two loading paradigms.** %1RM and RPE/RIR autoregulation. RPE 10 = max completable set; 9/8/7 = 1/2/3 RIR; 6 = 4+ RIR. RPE lets load adjust up/down to readiness. (RTS/Tuchscherer; PMC7810043; PMC8762534) — *high*
- **Specialty schemes only marginally beat straight sets.** Pooled advanced techniques (rest-pause, drop, cluster, tempo, VBT, eccentric) g=0.159 over straight sets; heterogeneous (strength g=0.351 vs hypertrophy g=0.046). ⇒ **straight sets = strong default**; use specialty schemes for power/strength niches + variety/fatigue/time. (Tsartsapakis 2026, PMC12922048) — *high*
- **Strength → straight sets (default).** Larger 1RM effect sizes than cluster in trained powerlifters (bench 2.56 vs 1.29, squat 1.55 vs 1.06, deadlift 2.16 vs 1.2). Rest-pause slightly > traditional squat 1RM; drop sets = traditional. Treat as modest default, not absolute (2025 meta found no TS-vs-CS strength difference). (Arazi 2021 Nature s41598-021-87372-y; Prestes 2021 PMID 34260860) — *high*
- **Power → cluster sets.** 20-30s intra-set rest. Wk1-4 2×5@70%→2×3@85%; Wk5-8 2×4@80-85%→2×2@95%, ~120-180s inter-set. CS lower-body peak power +17.1% (ES 1.15) vs TS +5.1%. Value = reducing per-set velocity/power loss. (Arazi 2021; PMC8041766) — *high*
- **Hypertrophy → drop / rest-pause / superset / variable resistance all ≈ straight** at equated volume — choose by fatigue/time/preference. Drop sets reach equal hypertrophy in ½–⅓ the time (higher fatigue). (Sodal 2023 PMC10390395; Prestes 2021; SportsMed 2025 s40279-025-02176-8; Fonseca 2023 PMC11022786) — *high*
- **Complex-contrast / PAP → power+strength.** Heavy ≥85% 1RM (88.2% of studies) alternated set-by-set with explosive bodyweight (77.9%) or ~30% 1RM (19.1%); ~180s intra-contrast rest; 3 sets; 2×/week; ≥48h recovery. Intensification/peak; heavy element = main lift/variation. (PMC11436572) — *high*
- **Accommodating resistance (bands/chains) → strength modifier, gated.** Only trained lifters at **≥80% 1RM** (ES 0.76); ES 0.00 at <80%. No hypertrophy advantage. Per-set modifier on main lifts in intensification. (PMC9317775) — *high*
- **Velocity-based → power autoregulation cutoff.** VBT ≈ PBT for squat 1RM (low-grade). **Velocity-loss 10-20%** cutoff for jump/power. (PMC8601436; Jukic 2022 PMC9399433) — *high (mapping), low-grade (equivalence)*

## Refuted (do NOT encode)

- Advanced techniques map "primarily to strength not hypertrophy" — **rejected** (1-2).
- Rest-pause/drop = traditional hypertrophy at equated volume stated as proven equivalence — **rejected as overclaim** (1-2); it's absence-of-difference, small samples.
- Variable resistance beats constant for strength **overall** ES 0.80 — **rejected**; benefit confined to trained ≥80%.
- VBT comparable strength at lower volume (77% reps) — **rejected** (0-3).
- Low velocity-loss VBT superior to all approaches for power — **rejected** (0-3).

## Coaching consensus (label "근거 약함" in UI — not RCT-verified here)

Pyramids (ascending/reverse), top-set+back-off, top-single+back-off, ramping to daily max, AMRAP/PR sets (Wendler 5/3/1 "+"), myo-reps, mechanical drop, wave loading (3-2-1), compound/giant/superset grouping, pre-exhaust, EMOM/density, widowmaker (20-rep), pin/dead-stop, tempo/paused as loading modifier, Prilepin's table. Anchor to Wendler, Nuckols/Stronger By Science, Helms/MASS, Israetel/RP, Tuchscherer/RTS, Schoenfeld, Juggernaut.

## Honest-limits to surface in UI

Most "equivalence" = absence of a detected difference at small n (8-10/group), not formal equivalence. Cluster-power and TS-strength advantages lean on one 8-wk RCT with wide CIs. VBT=PBT is low-grade. Accommodating-resistance benefit is trained + ≥80% only.
